HUBViewController

@protocol HUBViewController <NSObject>

Protocol defining the public API of a Hub Framework view controller

You don’t conform to this protocol yourself, instead the Hub Framework will create view controllers conforming to this protocol through HUBViewControllerFactory.

  • delegate

    The view controller’s delegate. See HUBViewControllerDelegate for more information.
