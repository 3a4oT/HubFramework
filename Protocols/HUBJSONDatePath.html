HUBJSONDatePath

@protocol HUBJSONDatePath <HUBJSONPath>

Protocol defining the API for a JSON path that points to an NSDate value

See HUBJSONPath and HUBMutableJSONPath for more information on how JSON paths work.

  • -dateFromJSONDictionary:

    Return an NSDate value by following this path in a JSON dictionary

    Declaration

    Objective-C

    - (nullable NSDate *)dateFromJSONDictionary:
    (nonnull NSDictionary<NSString *, NSObject *> *)dictionary;

    Swift

    func dateFromJSONDictionary(dictionary: [String : NSObject]) -> NSDate?

    Parameters

    dictionary

    The JSON dictionary to apply the path on

    Return Value

    An NSDate instance parsed from an NSString value found by following the path, or nil as the default in case an operation in the path failed, or if the found value wasn’t an NSString or couldn’t be parsed into an NSDate.
